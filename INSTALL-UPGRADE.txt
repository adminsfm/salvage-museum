HOW TO INSTALL THIS STAFF EDITOR UPGRADE

This folder contains three replacement/add-on files:

1. museum-editor-server.js
2. staff-editor.html
3. START-EDITOR.command

To install:

1. Open the museum website folder:
   salvage-fishermans-museum

2. Drag these three files into that folder.

3. If the computer asks whether to replace existing files, choose Replace.

4. Double-click START-EDITOR.command.

5. The editor should open at:
   http://localhost:4173/editor

Keep STAFF-GUIDE.txt with the website folder for future staff.

The public website pages are not changed by installing this editor upgrade.
This only changes the editing tool staff use.
