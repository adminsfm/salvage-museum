How to Use the Website Editor

This editor is meant for everyday museum updates: words, photos, pages, and simple sections.

Choose a page

Use the Website Pages list on the left. The selected page opens in the large preview area.

Edit and save

Click words in the preview and type. Use the top toolbar for bold, headings, links, photos, and saving.

Add sections

Use Add Section Template for text sections, photo boxes, notes, timeline items, and callout bands.

Add and crop photos

Click a photo box, then choose or drag a photo. Use Photo up/down/left/right to adjust the crop.

Change color safely

Click a box, then use Style Selected Box. The choices are limited to the museum color palette.

Publish later

Save changes here first. Then use GitHub Desktop to commit and push when you are ready.

  1. Pick a page from the left.
  2. Click the words or box you want to change.
  3. Add templates or photos only where needed.
  4. Click Save changes before leaving the page.
  5. Use GitHub Desktop to commit and push when the website is ready to update online.
